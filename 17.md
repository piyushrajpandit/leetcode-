# Letter Combinations of a Phone Number

## Problem
Given a string containing digits from 2-9 inclusive, return all possible letter combinations that the number could represent.

Mapping of digits to letters follows the telephone keypad:

| Digit | Letters |
|-------|---------|
| 2     | abc     |
| 3     | def     |
| 4     | ghi     |
| 5     | jkl     |
| 6     | mno     |
| 7     | pqrs    |
| 8     | tuv     |
| 9     | wxyz    |

---

## Approach

- Use **backtracking (recursion)**.
- For each digit:
  - Get its corresponding letters from mapping.
  - Try each letter.
  - Recursively move to the next digit.
- When the index reaches the length of digits, store the generated string.

Time Complexity:  
O(4^n)  
(Each digit can map to at most 4 letters)

Space Complexity:  
O(n) recursion stack (excluding output)
